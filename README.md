# serve

Alternative to "python -m http.server --bind 0 9000"

## Getting started

### Install

* Install golang ;)

* Install `serve`

```shell
$ go get github.com/loganstone/serve
```

* Run

```shell
$ serve
```

### First contact

Click! http://localhost:9000

### Usage of Serve

```shell
$ serve -h
usage of serve:
  -d string
    	directory to serve (default ".")
  -p int
    	port to listen on (default 9000)
```

## Key Features

- Serving directory to http.
- Logging.
